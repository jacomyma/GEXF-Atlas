eZ Components - TranslationCacheTiein

.. contents:: Table of Contents

Introduction
============

The TranslationCacheTiein component ties together the Translation component with the
Cache component. This component only implements one class,
ezcTranslationCacheBackend, which is a backend for the Translation component.
